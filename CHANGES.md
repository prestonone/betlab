# Bet Lab frontend adjustments

This package was refined from the original Figma Make export.

## Changes made

- Converted all pricing from British pounds to Nigerian naira.
- Replaced Starter/Pro/Elite with Daily Pass, Weekly Lab and Monthly Lab.
- Set preview prices to ₦1,000/day, ₦3,500/week and ₦10,000/month.
- Removed free-trial claims and changed calls to action to Lab Access.
- Replaced generic categories with Banker, Sure 2, Sure 5 and Rollover.
- Removed fictional named analysts and replaced them with an honest Bet Lab operating structure.
- Reworked the About page around a skilled lead football analyst and the actual platform workflow.
- Replaced invented member counts and performance marketing claims in the hero with product facts.
- Replaced fake testimonials with product-experience preview cards.
- Localised registration plan selection for Nigerian customers.
- Preserved the midnight navy and gold design direction.

## Important

All match data, results, charts and account data are still frontend mock data. They will be replaced when the Django API is connected.
